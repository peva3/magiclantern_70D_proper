holy_grail
==========

Author: peva3
License: GPL v2+
Summary: Holy Grail timelapse with smooth day-to-night exposure ramping

Automates day-to-night (holy grail) timelapses by smoothly ramping
ISO, shutter speed, and aperture between user-defined start and end
settings across a sequence of frames. Uses existing prop_request_change
and intervalometer infrastructure.
